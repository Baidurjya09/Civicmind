# 🎮 CivicMind Demo Interfaces

## 🏛 CONTROL ROOM (USE THIS FOR HACKATHON!)

**File:** `dashboard_control_room.py`

**Run:**
```bash
streamlit run demo/dashboard_control_room.py
```

**What it is:**
- Professional AI governance control room
- 3-panel layout (Control | City | Intelligence)
- Live system feed
- Manual controls (inject crisis, spawn rebel)
- Stress test mode
- Dark theme with animations
- Looks like NASA control room

**Why use this:**
- ✅ Most impressive visually
- ✅ Shows all 5 themes clearly
- ✅ Interactive (judges can control it)
- ✅ Dramatic (rebel alert is HUGE)
- ✅ Professional polish
- ✅ **THIS IS YOUR WINNING UI**
