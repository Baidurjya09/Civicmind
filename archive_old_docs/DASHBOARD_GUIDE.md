# 🎮 CivicMind Dashboard Guide

## 🚀 Quick Start

### Open Dashboard
```bash
streamlit run demo/dashboard_live.py
```

**URL**: http://localhost:8503

---

## ⚙️ Configuration (Sidebar)

### Recommended Settings for Demo:
- **Difficulty**: 8 (high chaos - makes it exciting!)
- **Simulation Length**: 20 weeks
- **Governance Strategy**: Surveillance State (most dramatic)
- **Force Rebel Spawn**: ✅ CHECK THIS (guarantees rebel appears)

### Click "🚀 START SIMULATION" to begin

---

## 📊 Dashboard Layout

### LEFT COLUMN - System Status
Shows the overall health of your city:

**System Health Score**
- 🟢 70%+ = Healthy
- 🟡 40-70% = Unstable
- 🔴 Below 40% = CRITICAL

**Key Metrics**
- 👥 Trust Score (how much citizens trust government)
- 💰 Budget (money remaining)
- ❤️ Survival Rate (% of citizens alive)
- ⚡ Civil Unrest (how angry citizens are)

**Rebel Status**
- Green = Safe
- Yellow = High risk (trust below 35%)
- 🚨 RED BLINKING = REBEL ACTIVE!

**Active Crises**
- Shows current disasters (floods, disease, strikes, etc.)

---

### CENTER COLUMN - Live Metrics

**Charts**
1. **Trust & Survival Over Time** - Shows how metrics change
2. **Reward Score** - Shows agent performance
3. **Economic Health (GDP)** - Shows economy

**Week Progress**
- Progress bar showing current week
- Week X / 20 counter

---

### RIGHT COLUMN - Agent Activity

**Latest Agent Decisions**
- Shows what each agent is doing
- Green border = normal decision
- Red border = conflict/bad decision
- ⚔️ icon = agent conflict

**Agent Conflicts**
- Shows when agents disagree
- Shows consequences of bad decisions

**Citizen Voices**
- 🔴 URGENT = Critical petition
- 🟡 Important = Medium priority
- 🟢 Normal = Low priority

---

## 🎮 Control Buttons

### ▶️ Next Week
- Advances simulation by 1 week
- Use this to go step-by-step
- Best for demo (judges can see details)

### ⏩ Fast Forward (5 weeks)
- Jumps ahead 5 weeks
- Use to speed up boring parts
- Use to get to rebel spawn faster

### 🔄 Reset
- Starts over from week 0
- Clears all history
- Use if demo goes wrong

### 💥 Trigger Crisis
- Instantly drops trust by 20%
- Increases unrest by 30%
- Forces system into critical state
- **USE THIS if rebel isn't spawning fast enough!**

---

## 🎯 Demo Flow (3 Minutes)

### Opening (30 seconds)
1. Show the dashboard
2. Point to 3 columns
3. Say: "6 AI agents governing a city in real-time"

### Setup (10 seconds)
- Show sidebar settings
- Point out "Force Rebel Spawn" checkbox
- Click "START SIMULATION"

### Action (1 minute 30 seconds)
1. Click "Next Week" 3-4 times slowly
2. Point to agent decisions appearing (right side)
3. Point to metrics changing (center)
4. Say: "Watch trust dropping, unrest rising"
5. Click "💥 Trigger Crisis" button
6. Click "Next Week" 5-6 more times
7. Point to conflicts appearing
8. Point to citizen petitions

### The Money Shot (45 seconds)
1. Keep clicking until **🚨 REBEL ACTIVE** appears
2. **STOP and point to the red blinking alert**
3. Say: "The rebel agent just spawned! This wasn't programmed to happen at a specific time - it emerged because the government failed."
4. Point to rebel strength percentage
5. Say: "This is our wild card - emergent agent behavior that no other project has."

### Close (15 seconds)
- Say: "CivicMind covers all 5 themes with this emergent rebel mechanic"
- Show the footer: "All 5 Themes | 6 Bonus Prizes"

## ⚠️ Troubleshooting

### Rebel Not Spawning?
1. Click "💥 Trigger Crisis" button
2. Click "Next Week" 3-4 times
3. Trust should drop below 30%
4. Rebel will spawn within 2-3 weeks

### Dashboard Too Slow?
- Use "⏩ Fast Forward" button
- Skip to interesting parts

### Made a Mistake?
- Click "🔄 Reset" button
- Start over

### Text Hard to Read?
- The new version has:
  - Larger fonts (16-20px)
  - Better contrast
  - Clearer labels
  - Bigger metrics
  - More spacing

## 💡 Pro Tips

### For Maximum Impact:
1. **Go slow** - Let judges see what's happening
2. **Point and explain** - Don't assume they understand
3. **Use the crisis button** - Guarantees drama
4. **Wait for rebel** - It's your best feature
5. **Show conflicts** - Proves multi-agent complexity
6. **Read petitions** - Shows personal theme

### What to Say:
- "Watch the agents making decisions"
- "See how they conflict with each other"
- "Notice trust is dropping"
- "Here comes a crisis"
- "And there's the rebel!"
- "This is emergent behavior"

### What NOT to Say:
- ❌ "I hope this works" (sounds uncertain)
- ❌ "Sometimes it crashes" (sounds broken)
- ❌ "The rebel might not spawn" (you have Force Rebel!)
- ❌ Technical jargon judges won't understand
