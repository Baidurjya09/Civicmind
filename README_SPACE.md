# CivicMind: Multi-Agent Governance Simulation

A reinforcement learning environment where 6 AI agents govern a city through crises.

## Features

- **6 Specialized Agents**: Each with unique objectives
- **Emergent Rebel Agent**: Spawns when governance fails
- **52-Week Episodes**: Long-horizon planning
- **Auto-Escalating Difficulty**: Self-improving challenges
- **Interactive Demo**: Try it yourself!

## How to Use

1. Configure the environment
2. Reset to start
3. Choose actions for each agent
4. Watch the city evolve!

## Training Results

- Q-Learning: +18.4% reward, +107% trust improvement
- LLM: Each agent learns unique policies

Built for Meta × Hugging Face OpenEnv Hackathon 2025.
